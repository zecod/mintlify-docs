## Get Your API Key

<Steps>
  <Step title="Create an Account">
    Sign up for a Pro or Business account at [neospeech.io](https://neospeech.io)

    <Warning>Free accounts do not have API access</Warning>
  </Step>

  <Step title="Generate API Key">
    Navigate to **API Settings** in your dashboard and generate a new API key

    Your API key format: `sk-` + 32 characters = 35 characters total
  </Step>

  <Step title="Secure Your Key">
    Store your API key securely. Never commit it to version control.

    ```bash
    # Add to your .env file
    NEOSPEECH_API_KEY=sk-your-api-key-here
    ```
  </Step>
</Steps>

## Make Your First Request

<CodeGroup>

```bash cURL
curl -X POST "https://api.neospeech.io/v1/audio/speech" \
  -H "Authorization: Bearer sk-your-api-key-here" \
  -H "Content-Type: application/json" \
  -d '{
    "input": "Hello, this is my first text-to-speech request!",
    "voice": "lyra",
    "model": "aurora-4"
  }' \
  --output speech.mp3
```

```javascript JavaScript
const apiKey = process.env.NEOSPEECH_API_KEY;

async function textToSpeech() {
  const response = await fetch('https://api.neospeech.io/v1/audio/speech', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      input: 'Hello, this is my first text-to-speech request!',
      voice: 'lyra',
      model: 'aurora-4'
    })
  });

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  const audioBlob = await response.blob();
  return audioBlob;
}

textToSpeech();
```

```python Python
import requests
import os

api_key = os.getenv('NEOSPEECH_API_KEY')

response = requests.post(
    'https://api.neospeech.io/v1/audio/speech',
    headers={
        'Authorization': f'Bearer {api_key}',
        'Content-Type': 'application/json'
    },
    json={
        'input': 'Hello, this is my first text-to-speech request!',
        'voice': 'lyra',
        'model': 'aurora-4'
    }
)

if response.status_code == 200:
    with open('speech.mp3', 'wb') as f:
        f.write(response.content)
    print('Audio saved successfully')
else:
    print(f'Error: {response.json()}')
```

</CodeGroup>

## Understanding the Request

### Required Parameters

<ParamField path="input" type="string" required>
  The text to convert to speech (max 5,000 characters)
</ParamField>

<ParamField path="voice" type="string" required>
  Voice identifier. Popular options: `lyra`, `kai`, `zara`

  [View all voices](/api-reference/voices)
</ParamField>

<ParamField path="model" type="string" required>
  Audio model. Options: `aurora-4`, `aurora-3.5`, `aurora-3`, `turbo-3`, `mini-2`

  [Learn about models](/concepts/models)
</ParamField>

### Optional Parameters

<ParamField path="pitch" type="string" default="+0%">
  Voice pitch adjustment (-50% to +50%)
</ParamField>

<ParamField path="style" type="string" default="calm">
  Speech style: `calm`, `excited`, `cheerful`, `sad`
</ParamField>

<ParamField path="styleDegree" type="string" default="1.5">
  Style intensity (0.5 to 2.0)
</ParamField>

## Understanding the Response

The API returns audio in MP3 format with metadata in response headers:

```
Content-Type: audio/mpeg
X-Character-Count: 47
X-Model-Used: aurora-4
X-Voice-Used: Lyra
```

## Try Streaming

For real-time audio generation, use the streaming endpoint:

```bash
curl -X POST "https://api.neospeech.io/v1/audio/stream" \
  -H "Authorization: Bearer sk-your-api-key-here" \
  -H "Content-Type: application/json" \
  -d '{
    "input": "This is a streaming test",
    "voice": "kai",
    "model": "turbo-3"
  }' \
  --output stream.mp3
```

<Info>
Streaming provides lower perceived latency by sending audio chunks as they're generated
</Info>

## Error Handling

Always check the response status and handle errors:

<CodeGroup>

```javascript JavaScript
try {
  const response = await fetch('https://api.neospeech.io/v1/audio/speech', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({ input, voice, model })
  });

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message || `HTTP ${response.status}`);
  }

  return await response.blob();
} catch (error) {
  console.error('TTS Error:', error.message);
  throw error;
}
```

```python Python
try:
    response = requests.post(
        'https://api.neospeech.io/v1/audio/speech',
        headers={
            'Authorization': f'Bearer {api_key}',
            'Content-Type': 'application/json'
        },
        json={'input': text, 'voice': voice, 'model': model}
    )

    if response.status_code != 200:
        error_data = response.json()
        raise Exception(error_data.get('message', f'HTTP {response.status_code}'))

    return response.content
except Exception as e:
    print(f'Error: {e}')
    raise
```

</CodeGroup>

## Common Errors

<AccordionGroup>
  <Accordion title="401 Unauthorized">
    **Problem:** Missing or invalid API key

    **Solution:** Verify your API key format (`sk-` + 32 characters) and check it's active in your dashboard
  </Accordion>

  <Accordion title="403 Forbidden - Free Plan">
    **Problem:** Free accounts cannot access the API

    **Solution:** Upgrade to Pro or Business plan
  </Accordion>

  <Accordion title="413 Payload Too Large">
    **Problem:** Text exceeds 5,000 character limit

    **Solution:** Split your text into smaller chunks
  </Accordion>

  <Accordion title="400 Bad Request">
    **Problem:** Invalid parameters (voice, model, or malformed request)

    **Solution:** Check parameter values against documentation
  </Accordion>
</AccordionGroup>
