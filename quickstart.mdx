Grab your API key from the [dashboard](https://neospeech.io) (Pro or Business plan required). Export it once so you can reuse it in every command:

```bash
export NEOSPEECH_API_KEY="sk-your-api-key"
```

## 1. Generate Speech

Turn text into audio and save the result as an MP3 file.

<CodeGroup>

```bash cURL
curl https://api.neospeech.io/v1/audio/speech \
  -H "Authorization: Bearer $NEOSPEECH_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "input": "Hello from NeoSpeech!",
    "voice": "lyra",
    "model": "aurora-4"
  }' \
  --output hello.mp3
```

```javascript JavaScript
import fs from 'node:fs/promises';

const response = await fetch('https://api.neospeech.io/v1/audio/speech', {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${process.env.NEOSPEECH_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    input: 'Hello from NeoSpeech!',
    voice: 'lyra',
    model: 'aurora-4'
  })
});

const audioBuffer = Buffer.from(await response.arrayBuffer());
await fs.writeFile('hello.mp3', audioBuffer);
```

```python Python
import os
import requests

response = requests.post(
    'https://api.neospeech.io/v1/audio/speech',
    headers={
        'Authorization': f"Bearer {os.environ['NEOSPEECH_API_KEY']}",
        'Content-Type': 'application/json'
    },
    json={
        'input': 'Hello from NeoSpeech!',
        'voice': 'lyra',
        'model': 'aurora-4'
    }
)

with open('hello.mp3', 'wb') as file:
    file.write(response.content)
```

```java Java
var client = java.net.http.HttpClient.newHttpClient();
var request = java.net.http.HttpRequest.newBuilder()
    .uri(java.net.URI.create("https://api.neospeech.io/v1/audio/speech"))
    .header("Authorization", "Bearer " + System.getenv("NEOSPEECH_API_KEY"))
    .header("Content-Type", "application/json")
    .POST(java.net.http.HttpRequest.BodyPublishers.ofString("""
      {
        \"input\": \"Hello from NeoSpeech!\",
        \"voice\": \"lyra\",
        \"model\": \"aurora-4\"
      }
    """))
    .build();

var response = client.send(request, java.net.http.HttpResponse.BodyHandlers.ofByteArray());
java.nio.file.Files.write(java.nio.file.Path.of("hello.mp3"), response.body());
```

</CodeGroup>

You now have `hello.mp3` with natural-sounding narration.

## 2. List Voices

Explore the voice library so you can pick the right tone and locale.

<CodeGroup>

```bash cURL
curl -s "https://api.neospeech.io/v1/voices/list?gender=female&search=lyra&limit=10" \
  -H "Authorization: Bearer $NEOSPEECH_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.neospeech.io/v1/voices/list?gender=female&search=lyra&limit=10', {
  headers: {
    Authorization: `Bearer ${process.env.NEOSPEECH_API_KEY}`
  }
});

const data = await response.json();
console.log(`Returned ${data.data.voices.length} voice(s)`);
```

```python Python
import os
import requests

response = requests.get(
    'https://api.neospeech.io/v1/voices/list?gender=female&search=lyra&limit=10',
    headers={'Authorization': f"Bearer {os.environ['NEOSPEECH_API_KEY']}"}
)

voices = response.json()
print(f"Returned {len(voices['data']['voices'])} voice(s)")
```

```java Java
var client = java.net.http.HttpClient.newHttpClient();
var request = java.net.http.HttpRequest.newBuilder()
    .uri(java.net.URI.create("https://api.neospeech.io/v1/voices/list?gender=female&search=lyra&limit=10"))
    .header("Authorization", "Bearer " + System.getenv("NEOSPEECH_API_KEY"))
    .GET()
    .build();

var response = client.send(request, java.net.http.HttpResponse.BodyHandlers.ofString());
System.out.println(response.body());
```

## 3. List Models

Check available audio models to balance quality and speed.

<CodeGroup>

```bash cURL
curl -s "https://api.neospeech.io/v1/models/list" \
  -H "Authorization: Bearer $NEOSPEECH_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.neospeech.io/v1/models/list', {
  headers: {
    Authorization: `Bearer ${process.env.NEOSPEECH_API_KEY}`
  }
});

const models = await response.json();
console.log(models.data.models.map(model => model.shortname));
```

```python Python
import os
import requests

response = requests.get(
    'https://api.neospeech.io/v1/models/list',
    headers={'Authorization': f"Bearer {os.environ['NEOSPEECH_API_KEY']}"}
)

models = response.json()
print([model['shortname'] for model in models['data']['models']])
```

```java Java
var client = java.net.http.HttpClient.newHttpClient();
var request = java.net.http.HttpRequest.newBuilder()
    .uri(java.net.URI.create("https://api.neospeech.io/v1/models/list"))
    .header("Authorization", "Bearer " + System.getenv("NEOSPEECH_API_KEY"))
    .GET()
    .build();

var response = client.send(request, java.net.http.HttpResponse.BodyHandlers.ofString());
System.out.println(response.body());
```

## 4. Check Balance

Verify credits before large jobs or scheduled runs.

<CodeGroup>

```bash cURL

  -H "Authorization: Bearer $NEOSPEECH_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.neospeech.io/v1/balance', {
  headers: {
    Authorization: `Bearer ${process.env.NEOSPEECH_API_KEY}`

});

console.log(balance.data.balance);
```
