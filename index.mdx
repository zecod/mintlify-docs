NeoSpeech API converts text into speech audio. Send text, get MP3 audio back.

## What you can do

- Convert up to 5,000 characters per request
- Choose from 50 voices in 90+ languages
- Select quality level with 5 audio models
- Stream audio in real-time
- Authenticate with API key

## Basic usage

```bash
curl https://api.neospeech.io/v1/audio/speech \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "input": "Hello world",
    "voice": "lyra",
    "model": "aurora-4"
  }' \
  --output speech.mp3
```

Three required parameters:
- `input` - Your text
- `voice` - Voice ID (see [voice list](/api/voices))
- `model` - Quality level (see [models](/api/models))

## Requirements

- Pro or Business plan ([upgrade](https://neospeech.io/pricing))
- API key from dashboard
- HTTPS requests only
