## API Key Authentication

NeoSpeech API uses API key authentication via Bearer tokens. All requests must include a valid API key in the Authorization header.

## Getting Your API Key

<Steps>
  <Step title="Sign Up">
    Create a Pro or Business account at [neospeech.io](https://neospeech.io)

    <Warning>Free accounts cannot generate API keys</Warning>
  </Step>

  <Step title="Navigate to Settings">
    Go to **Dashboard** → **API Settings**
  </Step>

  <Step title="Generate Key">
    Click **Generate New API Key** and copy it immediately

    <Info>API keys are only displayed once. Store it securely.</Info>
  </Step>
</Steps>

## API Key Format

```
sk-XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

- **Prefix:** `sk-`
- **Length:** 32 characters
- **Total:** 35 characters

<Note>Invalid format will result in `400 Bad Request`</Note>

## Making Authenticated Requests

Include your API key in the `Authorization` header with the `Bearer` prefix:

<CodeGroup>

```bash cURL
curl -X POST "https://api.neospeech.io/v1/audio/speech" \
  -H "Authorization: Bearer sk-your-api-key-here" \
  -H "Content-Type: application/json" \
  -d '{"input": "Hello", "voice": "lyra", "model": "aurora-4"}'
```

```javascript JavaScript
const response = await fetch('https://api.neospeech.io/v1/audio/speech', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${process.env.NEOSPEECH_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ input, voice, model })
});
```

```python Python
import requests

headers = {
    'Authorization': f'Bearer {os.getenv("NEOSPEECH_API_KEY")}',
    'Content-Type': 'application/json'
}

response = requests.post(
    'https://api.neospeech.io/v1/audio/speech',
    headers=headers,
    json={'input': text, 'voice': voice, 'model': model}
)
```

</CodeGroup>

## Security Best Practices

### Store Keys Securely

Never hardcode API keys in your source code. Use environment variables:

```bash .env
NEOSPEECH_API_KEY=sk-your-api-key-here
```

```javascript
// Good
const apiKey = process.env.NEOSPEECH_API_KEY;

// Bad - Never do this
const apiKey = 'sk-1234567890abcdef1234567890abcdef123';
```

### Keep Keys Private

<Warning>
- Never commit API keys to version control
- Do not share keys in public forums or repositories
- Add `.env` to your `.gitignore` file
</Warning>

```gitignore .gitignore
.env
.env.local
.env.*.local
```

### Rotate Keys Regularly

Regenerate your API keys periodically:

1. Generate a new key in your dashboard
2. Update your application configuration
3. Test the new key
4. Revoke the old key

### Use Different Keys Per Environment

Maintain separate API keys for development, staging, and production:

```bash
# Development
NEOSPEECH_API_KEY_DEV=sk-dev-key-here

# Production
NEOSPEECH_API_KEY_PROD=sk-prod-key-here
```

## Authentication Errors

### 401 Unauthorized

**Missing Authorization Header**
```json
{
  "success": false,
  "message": "Authorization header is missing or invalid."
}
```

**Solution:** Include the Authorization header with Bearer token

### 400 Bad Request

**Invalid API Key Format**
```json
{
  "success": false,
  "message": "Invalid API key format."
}
```

**Solution:** Verify your API key follows the format: `sk-` + 32 characters

### 403 Forbidden

**Disabled or Invalid API Key**
```json
{
  "success": false,
  "message": "Access denied: API key is disabled or not found."
}
```

**Solution:** Check if your API key is active in the dashboard

**Free Plan Restriction**
```json
{
  "success": false,
  "message": "Access denied. This API is only available for Pro and Business plans. Please upgrade your plan to access our services."
}
```

**Solution:** Upgrade to Pro or Business plan

## Testing Your Authentication

Verify your API key works correctly:

```bash
curl -X GET "https://api.neospeech.io/v1/balance" \
  -H "Authorization: Bearer sk-your-api-key-here"
```

**Success Response:**
```json
{
  "success": true,
  "data": {
    "balance": { ... },
    "plan": {
      "type": "Pro",
      "renewal_type": "monthly"
    }
  }
}
```
