## Voice Testing

### Listen to Samples

Always listen to voice samples before production:

```bash
curl -s "https://api.neospeech.io/v1/voices/list" \
  -H "Authorization: Bearer $NEOSPEECH_API_KEY" \
  | jq -r '.data.voices[] | "\(.name): \(.sample_url)"'
```

### A/B Test Voices

Test multiple voices with your audience:

```bash
for voice in lyra zara marcus; do
  curl -X POST "https://api.neospeech.io/v1/audio/speech" \
    -H "Authorization: Bearer $NEOSPEECH_API_KEY" \
    -H "Content-Type: application/json" \
    -d "{\
      \"input\": \"Your test message here\",\
      \"voice\": \"${voice}\",\
      \"model\": \"aurora-3.5\"\
    }" \
    --output "${voice}-test.mp3"
done
```

### Test with Your Content

Generate samples using your actual content:

```bash
curl -X POST "https://api.neospeech.io/v1/audio/speech" \
  -H "Authorization: Bearer $NEOSPEECH_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "input": "Welcome to our platform. This is a sample of how our content will sound. Please evaluate the voice quality and tone.",
    "voice": "lyra",
    "model": "aurora-3.5"
  }' \
  --output sample-review.mp3
```

## Voice Customization

### Pitch Adjustment

Adjust voice pitch to match your brand:

```bash
# Higher pitch (more energetic)
curl -X POST "https://api.neospeech.io/v1/audio/speech" \
  -H "Authorization: Bearer $NEOSPEECH_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "voice": "lyra",
    "pitch": "+15%",
    "input": "Welcome to our service!"
  }' \
  --output lyra-high-pitch.mp3

# Lower pitch (more authoritative)
curl -X POST "https://api.neospeech.io/v1/audio/speech" \
  -H "Authorization: Bearer $NEOSPEECH_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "voice": "marcus",
    "pitch": "-10%",
    "input": "This is an important announcement."
  }' \
  --output marcus-low-pitch.mp3
```

### Style Control

Apply emotional styles to voices:

```bash
# Excited style for promotional content
curl -X POST "https://api.neospeech.io/v1/audio/speech" \
  -H "Authorization: Bearer $NEOSPEECH_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "voice": "kai",
    "style": "excited",
    "styleDegree": "1.8",
    "input": "Check out our amazing new features!"
  }' \
  --output kai-excited.mp3

# Calm style for meditation content
curl -X POST "https://api.neospeech.io/v1/audio/speech" \
  -H "Authorization: Bearer $NEOSPEECH_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "voice": "zara",
    "style": "calm",
    "styleDegree": "1.2",
    "input": "Take a deep breath and relax."
  }' \
  --output zara-calm.mp3
```
