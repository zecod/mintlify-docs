## Overview

NeoSpeech offers 5 specialized audio models, each optimized for different quality, speed, and use case requirements. Choose the model that best balances audio fidelity with processing latency for your application.

## Model Selection Helper

Programmatic model selection:

```javascript
function selectModel(requirements) {
  const {
    priority,        // 'quality' | 'speed' | 'balanced'
    useCase,         // specific use case
    latencyTarget,   // target latency in ms
    bandwidth,       // 'high' | 'medium' | 'low'
    platform         // 'web' | 'mobile' | 'iot'
  } = requirements;

  // Priority-based selection
  if (priority === 'quality') {
    return 'aurora-4';
  }

  if (priority === 'speed') {
    return latencyTarget < 1000 ? 'turbo-3' : 'mini-2';
  }

  // Platform-based selection
  if (platform === 'iot' || platform === 'mobile') {
    return 'mini-2';
  }

  // Bandwidth-based selection
  if (bandwidth === 'low') {
    return 'mini-2';
  }

  // Use case-based selection
  const useCaseMap = {
    'broadcasting': 'aurora-4',
    'podcast': 'aurora-3.5',
    'audiobook': 'aurora-3.5',
    'elearning': 'aurora-3.5',
    'presentation': 'aurora-3.5',
    'chatbot': 'turbo-3',
    'ivr': 'turbo-3',
    'gaming': 'turbo-3',
    'social-media': 'aurora-3',
    'documentation': 'aurora-3'
  };

  return useCaseMap[useCase] || 'aurora-3.5';
}

// Examples
console.log(selectModel({ priority: 'quality' }));
// Output: 'aurora-4'

console.log(selectModel({ useCase: 'podcast' }));
// Output: 'aurora-3.5'

console.log(selectModel({ platform: 'mobile', bandwidth: 'low' }));
// Output: 'mini-2'

console.log(selectModel({ priority: 'speed', latencyTarget: 800 }));
// Output: 'turbo-3'
```

## Performance Considerations

### Latency

Average processing times (for 100 characters):

- **Aurora 4:** ~2.8 seconds
- **Aurora 3.5:** ~2.2 seconds
- **Aurora 3:** ~1.8 seconds
- **Turbo 3:** ~0.9 seconds (fastest)
- **Mini 2:** ~1.2 seconds

### Bandwidth Requirements

Streaming bandwidth needed:

- **Aurora 4:** ~192 kbps
- **Aurora 3.5:** ~96 kbps
- **Aurora 3:** ~160 kbps
- **Turbo 3:** ~64 kbps
- **Mini 2:** ~48 kbps (lowest)

### Cost Efficiency

All models cost the same per character, so choose based on quality and speed needs, not cost per request.

## Testing Models

Compare models with your content:

```javascript
async function compareModels(text, voice) {
  const models = ['aurora-4', 'aurora-3.5', 'aurora-3', 'turbo-3', 'mini-2'];
  const results = [];

  for (const model of models) {
    const startTime = Date.now();

    const audio = await generateSpeech({
      input: text,
      voice: voice,
      model: model
    });

    const latency = Date.now() - startTime;
    const size = audio.size;

    results.push({
      model,
      latency,
      size,
      sizeKB: Math.round(size / 1024)
    });

    console.log(`${model}: ${latency}ms, ${Math.round(size / 1024)} KB`);
  }

  return results;
}

// Test with your content
const comparison = await compareModels(
  'This is a test message to compare audio models.',
  'lyra'
);
```
