## Overview

Get detailed information about your account balance, including total credits, used credits, remaining credits, overage usage, and plan type. This endpoint helps you monitor usage and manage costs.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token with your API key: `Bearer sk-your-api-key-here`
</ParamField>

## Response

### Success Response

**Status Code:** `200 OK`

**Content-Type:** `application/json`

```json
{
  "success": true,
  "data": {
    "total_credits": 1000000,
    "used_credits": 247850,
    "remaining_credits": 752150,
    "overage_credits": 0,
    "overage_cost": 0.00,
    "plan_type": "pro",
    "billing_cycle": {
      "start_date": "2025-09-01T00:00:00Z",
      "end_date": "2025-10-01T00:00:00Z",
      "days_remaining": 15
    }
  }
}
```

## Examples

<CodeGroup>

```bash cURL
curl -X GET "https://api.neospeech.io/v1/balance" \
  -H "Authorization: Bearer sk-your-api-key-here"
```

```javascript JavaScript
const apiKey = process.env.NEOSPEECH_API_KEY;

async function getBalance() {
  const response = await fetch('https://api.neospeech.io/v1/balance', {
    method: 'GET',
    headers: {
      'Authorization': `Bearer ${apiKey}`
    }
  });

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  const result = await response.json();
  const { data } = result;

  console.log(`Plan: ${data.plan_type.toUpperCase()}`);
  console.log(`Remaining credits: ${data.remaining_credits.toLocaleString()}`);
  console.log(`Usage: ${((data.used_credits / data.total_credits) * 100).toFixed(1)}%`);
  console.log(`Days remaining: ${data.billing_cycle.days_remaining}`);

  return data;
}
```

```python Python
import requests
import os

def get_balance():
    response = requests.get(
        'https://api.neospeech.io/v1/balance',
        headers={
            'Authorization': f'Bearer {os.getenv("NEOSPEECH_API_KEY")}'
        }
    )

    if response.status_code == 200:
        result = response.json()
        data = result['data']

        print(f'Plan: {data["plan_type"].upper()}')
        print(f'Remaining credits: {data["remaining_credits"]:,}')
        print(f'Usage: {(data["used_credits"] / data["total_credits"]) * 100:.1f}%')
        print(f'Days remaining: {data["billing_cycle"]["days_remaining"]}')

        return data
    else:
        error = response.json()
        raise Exception(error.get('message', 'Request failed'))
```

## Usage Monitoring

### Check Before Requests

Always check remaining credits before large batch operations:

```javascript
async function safeGenerateSpeech(text) {
  const balance = await getBalance();

  // Estimate credits needed (roughly 1 credit per character)
  const estimatedCredits = text.length;

  if (balance.remaining_credits < estimatedCredits) {
    throw new Error(
      `Insufficient credits: ${balance.remaining_credits} remaining, ` +
      `${estimatedCredits} needed`
    );
  }

  // Proceed with generation
  return await generateSpeech(text);
}
```

### Set Up Alerts

Monitor usage and receive alerts when approaching limits:

```javascript
async function checkBalanceAlert(threshold = 0.8) {
  const balance = await getBalance();
  const usagePercent = balance.used_credits / balance.total_credits;

  if (usagePercent >= threshold) {
    console.warn(
      `Warning: ${(usagePercent * 100).toFixed(1)}% of credits used. ` +
      `${balance.remaining_credits.toLocaleString()} credits remaining.`
    );

    // Send alert notification
    await sendAlert({
      level: usagePercent >= 0.9 ? 'critical' : 'warning',
      message: `Credit usage at ${(usagePercent * 100).toFixed(1)}%`,
      remaining: balance.remaining_credits
    });
  }

  return balance;
}
```

### Calculate Costs

Track overage costs for budget management:

```python
def calculate_costs(balance_data):
    """Calculate current and projected costs"""
    overage_rate = 0.00001  # $0.00001 per credit

    base_cost = {
        'free': 0,
        'pro': 49,
        'business': 299
    }

    plan = balance_data['plan_type']
    overage = balance_data['overage_credits']
    overage_cost = overage * overage_rate
    total_cost = base_cost.get(plan, 0) + overage_cost

    print(f'Plan cost: ${base_cost.get(plan, 0):.2f}')
    print(f'Overage cost: ${overage_cost:.2f}')
    print(f'Total cost: ${total_cost:.2f}')

    return {
        'base': base_cost.get(plan, 0),
        'overage': overage_cost,
        'total': total_cost
    }
```

## Best Practices

<AccordionGroup>
  <Accordion title="Monitor Usage Regularly">
    Check balance daily or before large operations to avoid service interruptions
  </Accordion>

  <Accordion title="Implement Credit Warnings">
    Set up alerts at 80% and 90% usage thresholds to prevent unexpected overage
  </Accordion>

  <Accordion title="Cache Balance Data">
    Cache balance responses for 5-10 minutes to reduce API calls
  </Accordion>

  <Accordion title="Plan Ahead">
    Monitor days remaining in billing cycle and plan credit-intensive operations accordingly
  </Accordion>

  <Accordion title="Track Overage">
    Keep detailed logs of overage usage to optimize costs and plan upgrades
  </Accordion>
</AccordionGroup>
