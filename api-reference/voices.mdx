## Overview

Get a paginated list of all available text-to-speech voices. Filter by gender, locale, or search by name. Access 45+ high-quality voices across multiple languages and accents.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token with your API key: `Bearer sk-your-api-key-here`
</ParamField>

## Query Parameters

<ParamField query="gender" type="string">
  Filter voices by gender

  **Options:** `male`, `female`, `neutral`
</ParamField>

<ParamField query="locale" type="string">
  Filter voices by language locale

  **Examples:** `en-US`, `en-GB`, `es-ES`, `fr-FR`, `de-DE`, `ja-JP`, `ko-KR`
</ParamField>

<ParamField query="search" type="string">
  Search voices by name or description

  Case-insensitive partial matching
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Number of results per page

  **Range:** 1 to 100
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of results to skip for pagination

  **Minimum:** 0
</ParamField>

<CodeGroup>

```bash cURL - All Voices
curl -X GET "https://api.neospeech.io/v1/voices/list" \
  -H "Authorization: Bearer sk-your-api-key-here"
```

```bash cURL - Filtered

  -H "Authorization: Bearer sk-your-api-key-here"
```

```javascript JavaScript
const apiKey = process.env.NEOSPEECH_API_KEY;

async function listVoices(filters = {}) {
  const params = new URLSearchParams(filters);
  const url = `https://api.neospeech.io/v1/voices/list?${params}`;

  const response = await fetch(url, {
    method: 'GET',
    headers: {
      'Authorization': `Bearer ${apiKey}`
    }
  });

  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }

  const result = await response.json();
  return result.data;
}

// Get all voices
const allVoices = await listVoices();
console.log(`Found ${allVoices.pagination.total} voices`);

// Filter by gender and locale
const femaleUSVoices = await listVoices({
  gender: 'female',
  locale: 'en-US'
});

// Search by name
const searchResults = await listVoices({
  search: 'professional'
});
```

```python Python
import requests
import os

def list_voices(filters=None):
    params = filters or {}
    response = requests.get(
        'https://api.neospeech.io/v1/voices/list',
        headers={
            'Authorization': f'Bearer {os.getenv("NEOSPEECH_API_KEY")}'
        },
        params=params
    )

    if response.status_code == 200:
        result = response.json()
        return result['data']
    else:
        error = response.json()
        raise Exception(error.get('message', 'Request failed'))

# Get all voices
all_voices = list_voices()
print(f'Found {all_voices["pagination"]["total"]} voices')

# Filter by gender and locale
female_us_voices = list_voices({
    'gender': 'female',
    'locale': 'en-US'
})

# Search by name
search_results = list_voices({
    'search': 'professional'
})
```

## Pagination Example

Retrieve all voices with pagination:

<CodeGroup>

```javascript JavaScript
async function getAllVoices() {
  const allVoices = [];
  let offset = 0;
  const limit = 50;

  while (true) {
    const result = await listVoices({ limit, offset });
    allVoices.push(...result.voices);

    console.log(`Retrieved ${allVoices.length} of ${result.pagination.total}`);

    if (allVoices.length >= result.pagination.total) {
      break;
    }

    offset += limit;
  }

  return allVoices;
}
```

```python Python
def get_all_voices():
    all_voices = []
    offset = 0
    limit = 50

    while True:
        result = list_voices({'limit': limit, 'offset': offset})
        all_voices.extend(result['voices'])

        print(f'Retrieved {len(all_voices)} of {result["pagination"]["total"]}')

        if len(all_voices) >= result['pagination']['total']:
            break

        offset += limit

    return all_voices
```
